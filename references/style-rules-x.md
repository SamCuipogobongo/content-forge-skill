## DO
- Start with a strong opinion or counter-intuitive data hook
- One core point per tweet, high information density
- Use specific numbers and cases instead of vague descriptions
- Thread structure: hook → core insight → practical advice → CTA
- Use emoji sparingly as visual separators
- Last tweet: summary or call-to-action

## DON'T
- No more than 5 tweets per thread
- No AI filler phrases ("it's worth mentioning", "in summary", etc.)
- Don't start every tweet with a number — vary sentence structure
- Max 2 hashtags, placed in the last tweet only
- Don't write like a textbook — write like sharing with a friend
